# nmobjdump

Personal implementation of nm and objdump program from linux system. 

## Installation && Tools

Just need to run 'make' on linux system.</br>

## Usage

./my_nm [INPUT_FILE]</br>
./myobjdump [INPUT_FILE]</br>
